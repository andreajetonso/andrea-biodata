# Andrea Jetonso - Biodata Website

A beautiful, modern 2-page website showcasing personal biodata with smooth animations and interactive elements.

## Features

- **Two Pages:**
  - **Page 1 (Home):** Profile picture with short description
  - **Page 2 (Biodata):** Complete biodata information

- **Animations & Interactions:**
  - Smooth page transitions
  - Floating profile image animation
  - Hover effects on all interactive elements
  - Scroll-triggered animations
  - Parallax background effects
  - Ripple effects on button clicks
  - Gradient animations

- **Responsive Design:**
  - Works perfectly on desktop, tablet, and mobile devices

## Setup Instructions

1. **Add Your Profile Picture:**
   - Place your profile picture in the same folder as `index.html`
   - Name it `profile.jpg` (or update the image source in `index.html`)
   - Supported formats: JPG, PNG, WebP

2. **Upload to Web:**
   - Upload all files (`index.html`, `styles.css`, `script.js`, and `profile.jpg`) to your web hosting
   - All files should be in the same directory
   - Access your website through your domain
